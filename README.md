# Web projekt - 2024 január

Készítette: [Nagy Zétény](https://znagy.hu)

## Telepítés:

### PNPM telepítése:

```bash
npm install -g pnpm
```

### Függőségek telepítése:

```bash
pnpm install
```

### Dev szerver indítása:

```bash
pnpm dev
```
